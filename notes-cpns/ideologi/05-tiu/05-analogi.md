# TIU — Analogi

Soal analogi mengharuskan menemukan **hubungan** antara pasangan kata, lalu menerapkan hubungan yang sama pada pasangan lain.

---

## Jenis Hubungan Analogi

| Jenis | Contoh |
| --- | --- |
| Sinonim | OTONOMI : MANDIRI = CERDAS : BANYAK AKAL |
| Antonim | GUNDUL : RAMBUT = BUGIL : PAKAIAN |
| Bagian dari | BURITAN : KAPAL = EKOR : BURUNG |
| Bahan → Produk | KAYU : PAPAN = BERAS : NASI GORENG |
| Alat → Fungsi | PISTOL : TEMBAK = PULPEN : TULIS |
| Profesi → Objek | GURU : MURID = DOKTER : PASIEN |
| Sebab → Akibat | JATUH : SAKIT = MENGANTUK : TIDUR |
| Satuan → Kelipatan | DETIK : MENIT = HARI : MINGGU |
| Sumber → Produk | HEWAN : SUSU = POHON : BUAH |
| Tempat → Fungsi | PUSKESMAS : SEHAT = SEKOLAH : PANDAI |
| Ilmu → Objek | ASTROLOGI : RAMALAN = PATOLOGI : PENYAKIT |

---

## Strategi Mengerjakan

1. **Tentukan hubungan** dari pasangan kata yang diberikan secara eksplisit
2. **Uji setiap pilihan** dengan pola hubungan yang sama
3. **Waspadai jebakan**: pasangan yang mirip tapi hubungannya beda arah atau beda jenis

---

## Catatan Penting

- **Urutan penting**: GURU : MURID ≠ MURID : GURU
- **Gunakan pola kalimat**: "A [hubungan] B" → cari "C [hubungan yang sama] D"
- **Satuan waktu**: detik < menit < jam < hari < minggu < bulan < tahun
- **Bendera negara**: Merah Putih (Indonesia), Hitam Merah Kuning (Jerman), Merah Putih Merah (Kanada), Biru Putih Merah (Perancis)
- **Indera**: mata (penglihatan), telinga (pendengaran), hidung (penciuman), lidah (pengecapan), kulit/tangan (perabaan)
