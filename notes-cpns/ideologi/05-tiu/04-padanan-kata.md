# TIU — Padanan Kata (Sinonim)

Soal padanan kata mengharuskan memilih kata lain yang **maknanya paling dekat** dengan kata yang dicetak kapital.

---

## Strategi Mengerjakan

1. **Tentukan makna inti** kata kapital terlebih dahulu
2. **Eliminasi pilihan** yang jelas berbeda makna
3. Pilih yang **paling tepat**, bukan sekadar mirip bunyi

## Jebakan Umum

| Kata | Jebakan | Alasan |
| --- | --- | --- |
| TENDENSI | Tekanan / Otot | Mirip kata "tendon" (urat/otot) tapi beda asal |
| KISI-KISI | Tabel / Pola kerja | Makna lazim di ujian ≠ makna sehari-hari |
| AMBIGUITAS | Kebingungan | Kebingungan = efeknya, ketidakjelasan = maknanya |
| DEPORTASI | Pengangkatan | "De-" bukan berarti naik, tapi keluar/usir |
| SEREBRUM | Otak kecil | Otak kecil = serebelum (cerebellum), bukan serebrum |
| KLEPTOFOBIA | Penyakit suka mencuri | Itu kleptomania; kleptofobia = takut kecurian |
| SUKSESI | Keberhasilan | Jangan disamakan dengan "sukses" — beda makna |
| PREMI | Uang iuran | Uang iuran = konteks asuransi; makna utama KBBI = uang hadiah |
| DELUSI | Kekecewaan | Delusi = keyakinan salah tentang kenyataan, bukan perasaan kecewa |
| GENERIK | Obat | Obat generik hanyalah contoh; makna inti = umum/tidak bermerek |
| EPILOG | Prolog | Prolog = pembukaan; epilog = penutup — keduanya berlawanan |
| KONKAF | Konveks | Konkaf = cekung; konveks = cembung — jangan tertukar |
| DEDUKSI | Induksi | Deduksi = umum→khusus; induksi = khusus→umum |
| FUTURISTIS | Teknologi tinggi | Futuristis = canggih/berorientasi depan, bukan sekadar teknologi tinggi |
| NABATI | Hewani | Nabati = dari tumbuhan; hewani = dari hewan — keduanya berlawanan |
| AMBIGUITAS | Ambisi | Mirip bunyi tapi beda total: ambiguitas = makna ganda; ambisi = hasrat kuat |
| MUTAKHIR | Permanen | Mutakhir = terbaru/masa kini; permanen = tetap/abadi |
| EVOKASI | Evakuasi / Emigrasi | Evokasi = menggugah rasa; evakuasi = pengungsian; emigrasi = pindah negara |

## Tips Kata Serapan

Banyak kata sulit berasal dari bahasa asing:

- **Selebrasi** ← celebration (Inggris) = perayaan
- **Tendensi** ← tendency (Inggris) = kecenderungan
- **Ambiguitas** ← ambiguity (Inggris) = ketidakjelasan
- **Virtual** ← virtual (Inggris) = maya
- **Deportasi** ← deportation (Inggris) = pengusiran

> Kalau ragu, coba ingat kata aslinya dalam bahasa Inggris — biasanya lebih mudah menebak maknanya.
